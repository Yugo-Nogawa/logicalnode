# Logic Tree

シンプルで集中できるロジックツリー作成ツール。Tree2にインスパイアされたミニマルなデザイン。

## 特徴

- キーボード操作中心の直感的なインターフェース
- ノイズのないシンプルなUI
- 高速な思考の展開をサポート
- JSON形式でのデータ保存

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. アプリケーションの起動

```bash
npm start
```

## キーボードショートカット

### ノード操作
- **Enter**: 子ノードを追加
- **Shift + Enter**: 兄弟ノードを追加
- **Ctrl + Delete**: ノードを削除

### ナビゲーション
- **Tab**: 子ノードへ移動
- **Shift + Tab**: 親ノードへ移動
- **↑ / ↓**: 兄弟ノード間を移動

### ファイル操作
- **Ctrl/Cmd + N**: 新規ファイル
- **Ctrl/Cmd + O**: ファイルを開く
- **Ctrl/Cmd + S**: ファイルを保存
- **Ctrl/Cmd + Q**: アプリケーションを終了

## ビルド

実行可能ファイルを作成するには：

```bash
npm run build
```

ビルドされたファイルは `dist` フォルダに生成されます。

## ファイル構造

```
logicalnode3/
├── main.js          # Electronメインプロセス
├── renderer.js      # レンダラープロセス（ツリーロジック）
├── index.html       # UIテンプレート
├── styles.css       # スタイリング
├── package.json     # プロジェクト設定
└── README.md        # このファイル
```

## 使い方

1. アプリケーションを起動すると、最初のノードが自動的に作成されます
2. テキストを入力してアイデアを記述
3. Enterで子ノード、Shift+Enterで兄弟ノードを追加
4. Tab/Shift+Tabや矢印キーでノード間を移動
5. Ctrl+Sでツリーを保存
