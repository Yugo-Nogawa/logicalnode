# Logical Node 3 - Electron Edition

デスクトップ専用版（Windows/macOS/Linux）

## 特徴

- ✅ Windows/macOS/Linux対応
- ✅ ネイティブファイルシステム
- ✅ 安定した実績

## セットアップ

```bash
npm install
```

## 開発

```bash
npm start
```

## ビルド

```bash
# すべてのプラットフォーム
npm run build

# Windows のみ
npm run build:win

# macOS のみ
npm run build:mac

# Linux のみ
npm run build:linux
```

## ファイルサイズ

約50-100MB

## 推奨

より軽量な Tauri版（3-10MB）も検討してください：
```
../tauri/
```
